# 家庭内部点餐微信小程序（框架）

这个目录提供一个可直接作为微信小程序项目导入的基础框架，满足：

- 家庭成员选择点餐人、用餐时间、菜单并提交。
- 主账号后台查看汇总结果（按菜品聚合 + 成员明细）。
- 预留菜品名称、图片、做法字段，便于你后续自行补充。

## 目录结构

```txt
wechat-family-ordering-miniapp/
├── app.js
├── app.json
├── app.wxss
├── pages/
│   ├── order/            # 家庭成员点餐页
│   └── admin/            # 主账号后台汇总页
├── utils/
│   ├── menu.js           # 菜单数据（你后续补充）
│   └── order-service.js  # 提交/读取订单服务（本地 mock，可替换云开发）
└── cloudfunctions/
    └── submitOrder/      # 云函数占位（可接入正式后端）
```

## 你需要补充的内容

1. 在 `utils/menu.js` 中补充真实菜品：
   - `name`: 菜名
   - `image`: 菜品图片地址
   - `recipe`: 制作方法
2. 若要多人实时共享数据，把 `utils/order-service.js` 从本地存储改成：
   - 微信云开发数据库（推荐）
   - 或你的私有后端 API
3. 在 `pages/admin/admin.js` 的 `MAIN_ACCOUNT` 改为你的主账号身份字段（如昵称、openid）。

## 运行方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择 `wechat-family-ordering-miniapp`。
4. 根据需要配置 AppID（测试可使用测试号）。
